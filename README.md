# php-sillabe
Una serie di funzioni php per gestire la sillabazione in lingua italiana.
L'argomento è piuttosto complesso e a volte poco deterministico.

Intendo in futuro trasformare le varie funzioni in una classe e migliorare le funzioni e il loro ordine.
Per ora si tratta di un porting da una serie di funzioni javascript in parte imprecise.
